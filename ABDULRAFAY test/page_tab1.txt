H2: HTML Documents
P: All HTML documents must start with a document type declaration: . &lt;!DOCTYPE html&gt;
P: The HTML document itself begins with and ends with . &lt;html&gt; &lt;/html&gt;
P: The visible part of the HTML document is between and . &lt;body&gt; &lt;/body&gt;
H3: Example
H2: The &lt;!DOCTYPE&gt; Declaration
P: The declaration represents the document type, and helps browsers to display web pages correctly. &lt;!DOCTYPE&gt;
P: It must only appear once, at the top of the page (before any HTML tags).
P: The declaration is not case sensitive. &lt;!DOCTYPE&gt;
P: The declaration for HTML5 is: &lt;!DOCTYPE&gt;

H2: How to View HTML Source
P: Have you ever seen a Web page and wondered &quot;Hey! How did they do that?&quot;
H3: View HTML Source Code:
P: Click CTRL + U in an HTML page, or right-click on the page and select &quot;View Page Source&quot;. This will open a 
new tab 
containing the HTML source code of the page.
H3: Inspect an HTML Element:
P: Right-click on an element (or a blank area), and choose &quot;Inspect&quot; to see what elements are made up of (you will see both 
the HTML and the CSS). You can also edit the HTML or CSS on-the-fly in the 
Elements or Styles panel that opens.
